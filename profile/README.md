## About

**Notchly** is a lightweight native macOS app that gives the MacBook notch something useful to do.

It adds custom HUDs, live activities, lock screen support, swipe gestures, smooth transitions, and small native interactions around the notch, making it feel like an intentional part of macOS instead of unused screen space.

Notchly is built to stay simple, fast, and focused.

## Features

| Feature | What it adds |
| --- | --- |
| Custom HUDs | Compact overlays designed for the notch area. |
| Live activities | Glanceable status and activity updates where your eyes already look. |
| Lock screen support | Useful notch interactions beyond the main desktop. |
| Swipe gestures | Quick, natural controls without interrupting your flow. |
| Smooth transitions | Native-feeling motion built with SwiftUI. |
| Lightweight design | A focused macOS utility without unnecessary bloat. |
