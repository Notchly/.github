## About

**Notchly** is a lightweight native macOS app that turns your MacBook notch into a useful and interactive space.

It adds custom HUDs, live activities, lock screen support, swipe gestures, smooth transitions, and small native-feeling interactions that make the notch feel like part of macOS — not just unused screen space.

Notchly is built to stay simple, fast, and focused without unnecessary bloat.
